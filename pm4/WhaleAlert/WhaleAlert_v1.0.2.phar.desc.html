General

WhaleAlert in PocketMine! This plugin is for alerts if there is a large amount of Pay Transactions Economys

Commands

Commands Toggle for On/Off WhaleAlert Default if player Joined is On

Commands Aliases Default Permission
/whalealert /whalealert True whalealert.toggle

Feature

  • Alert if Transaction Minimum Reached
  • Multiple Economys
  • Can Auto detect Economys Plugin
  • Customable Minimum
  • Customable Broadcast Message
  • Broadcast to Discord can Embeds or Not

Config

---
# Your Economy plugin name

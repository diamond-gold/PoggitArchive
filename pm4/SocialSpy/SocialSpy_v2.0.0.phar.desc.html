SocialSpy is a plugin who allow you to spy players commands, the commands are also logged into the console If you have any issues you can report them at : https://github.com/superbobby2000/SocialSpy/issues

Commands

  • /socialspy or /ss (turn on socialspy)

Permission

  • socialspy.command (allow you to use socialspy commands)
  • socialspy.hide (hide the commands you use from players who have SocialSpy on)

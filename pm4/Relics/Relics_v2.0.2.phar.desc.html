General

This plugin is an easy to use relic plugin for your server. It allows players to find "relics" when mining that can give any reward you want! It is super easy to use and can be customised to your hearts content! Check the Installing tab to get started!

Installing

To install this plugin, you must first download the .phar. Once you have the .phar you must navigate to your plugins folder. Now just place the .phar inside this folder and restart your server.

Configuring

To configure this plugin, you first need to navigate to your plugin_data folder. Then, you need to find the Relics folder. Once in this folder you must open the config.yml file. Now just edit whatever you want!

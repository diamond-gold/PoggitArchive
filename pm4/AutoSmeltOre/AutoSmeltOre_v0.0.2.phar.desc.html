AutoSmeltOre

NOTICE: This plugin branch is for PocketMine-MP 4.

Auto smelt ore when break!

Features

  • Auto smelt ore when break!
  • Simple to use

How to Install

  1. Download the latest version
  2. Place the AutoSmeltOre.phar file into the plugins folder.
  3. Restart the server.
  4. Done!

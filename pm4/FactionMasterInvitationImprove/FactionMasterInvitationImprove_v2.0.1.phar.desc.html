FactionMaster Invitation systemm

This plugin is part of FactionMaster and requires installation on the server, see here for more information

Adds a panel between entering the field to send an invitation, and sending it. This panel will allow you to have several choices if the nickname/name entered is not complete. Works with players and factions registered on the server by the plugin.

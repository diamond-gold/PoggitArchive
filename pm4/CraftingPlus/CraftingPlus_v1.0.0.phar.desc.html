CraftingPlus

A PocketMine-MP plugin for adding custom recipes

Features

  • Support shaped crafts.
  • Support shapeless crafts.
  • Support smelting.
  • Lightweight and open source ❤️

Upcoming Features

  • Currently none planned. You can contribute or suggest for new features.

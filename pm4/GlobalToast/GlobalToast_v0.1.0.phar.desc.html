Description

  • This plugin allows you to broadcast toast announcements to everyone in the server!
  • You can modify the title broadcast by changing it on the config! Make sure to take a look!
  • Heres my YouTube channel if you want to check the plugin review: https://www.youtube.com/watch?v=7kSuConazTI

How To Use It

  • The main command is /globaltoast
  • Example: /globaltoast This is a test message!!

Permissions

  • globaltoast.cmd: Allows you to use the command!

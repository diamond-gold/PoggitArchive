BreakWarn

This plugin will warn player before tool in hand get break. The warn is toggleable, and users can switch between warn arts.

Command-ModeUsers with permissionbreakwarn.allowcan use/breakwarnand receive warnings

Tool-ModeUsers with permissionbreakwarn.toolcan use/breakwarnand upgrading Tools to BreakWarn Tools

Alert types ☰

  • Chat
  • Popup
  • Screen (as title)
  • All (All together)

Features ✓

  • [x] Switch language via config
  • [x] All output messages are customizable
  • [x] Different messages for different tooltypes
  • [x] Switchable warntypes (chat,popup,screen,all)
  • [x] Warn is toggleable
  • [x] BreakWarn per Tool (Tools can have BreakWarn)
  • [x] BreakGuard Option (Make tool useless before breaks)
  • [x] BreakWarn Tools have an enchantment

BreakWarn

This plugin will warn player before tool in hand get break.
The warn is toggleable, and users can switch between warn arts.
Two modes (command,tool) available.

Command mode = users with permission breakwarn.allow can use /breakwarn command and receive warnings
(chat,popup,screen,all)

Tool mode = user with permission breakwarn.tool can use /breakwarn and upgrading Tools to BreakWarn Tools
(chat,popup,screen,all)

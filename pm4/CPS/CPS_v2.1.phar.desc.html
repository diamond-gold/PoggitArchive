Features

CPS Popup -- Can be disabled/enabled with /cps <on/off>

CPS Limiter -- Max CPS can be edited through the config.

CPS Alerts -- Alerts all players with the "cps.alerts" permission of people clicking more CPS than the set amount in the config.

Discord Webhook -- Sends a Webhook to the URL set in the config.yml with the settings your define.

Intro

EasyCommandAutofill is a plugin which adds type-specified command arguments to MineCraft's autofill by parsing command usage messages. By adding this plugin to your server, all command usages written in the vanilla style will be parsed and sent to MineCraft with autofill enabled.

Features

  • Enables autofill for commands on your server
    • Simple argument support
    • Custom argument type support
    • Multiple argument trees
  • Multi-language support using PocketMine's language API
  • Configurable Debug-command highlighting

Future Additions

  • Parsing support for usage messages which do not follow vanilla style
  • More interactions with PlayerSelectors

This plugin is used to sudo players!

It's a simple plugin, check examples/commands for more information.

Commands:

  • /sudo player message/command - Send a message or run a command as another player!

Examples:

  • /sudo ErikX Hi There!
  • /sudo ErikX /say HI there!

Permissions:

Permission Default Description
sudo.use OP Permission to sudo a player!

FAQ

/tell isn't working.

It's working, this is the correct way to use it, /sudo Erik1206 /tell steve hey!

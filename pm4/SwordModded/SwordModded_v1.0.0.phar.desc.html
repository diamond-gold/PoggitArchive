Description

With the plugin you can modify the damage and durability of swords

Use

To change the damage en durability of the swords go to the configuration locate in plugin_data folder

Config

golden_durability: "32" golden_attack: "4" iron_durability: "250" iron_attack: "6" wooden_durability: "59" wooden_attack: "2" stone_durability: "131" stone_attack: "3" diamond_durability: "1561" diamond_attack: "8"

EconomyCShop

Chest Shop plugin for EconomyAPI. Non-Op players can sell their items inside a chest safely. While Chest Shop is open, no players except owner can open the chest inventory.

Usage

How to make Chest Shop

  1. Place chest and add items you want to sell.
  2. Place sign on chest (wall sign) with format below.
    1. Line 1: (none)
    2. Line 2: price:int
    3. Line 3: amount:int
    4. Line 4: item:string(item name) or (id:int):(meta:int)
  3. Complete!

How to use Chest Shop

just tap shop sign.

Config

  • repository decides which type of datasource is used. json or sqlite (strongly recommend you to use json by default)

Permission

  • economy-c-shop.force-close-shop allow to destroy others' shop.

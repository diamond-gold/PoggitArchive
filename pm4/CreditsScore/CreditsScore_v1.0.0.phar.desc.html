CreditsScore

A Simple ScoreHud Addon that allows you to use {credits} in the config with the Credits by LaithDev

How does it work?

Its very simple, go into your scorehud plugin_data folder and find ScoreHud.yml, then go to the line which holds the configurable scoreboard then add {credits}, example:

- "Credits: {credits}"

What if i catch an issue?

Report the issue in github by the issues tab u can do that by clicking this link: https://github.com/ItsToxicGG/CreditsScore/issues

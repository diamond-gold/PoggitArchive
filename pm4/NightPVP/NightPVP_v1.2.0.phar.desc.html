🌠 NightPvP

🌌NightPVP is plugin makes it so pvp can only happen at night.

❗ Requirements

PocketMine-MP API 4.0.0 and PHP 8 or PHP 7

❓ How to use

Put the worlds that have PvP enabled at night Put the world that will have PvP enabled at night in config.yml

ConsoleExporter


This plugin is useful for creating test images of console-based plugins.
It records the console and outputs it as an html file like above.
You can download it as a png through the controls button of the html file.

Usage

⚡ Usage

  1. Type /consoleexporter(or /ce) to start recoding.
  2. Do anything you want to record.
  3. Retype /consoleexporter(or /ce) to stop recording.
  4. The result is output as an HTML file.

✨ Advanced usage:

annotation:

Type anything starts with #.
It will be not executing as command.

line-break:

If you type only #.
It will be replace to empty line.

export png:
  1. Open the output HTML file with a browser.
  2. Click the 3 buttons at the top left of the page.
  3. The PNG file will be downloaded.

Command and Permission

commands:
  • name: consoleexport
  • alias: ce
  • description: Start console recording / Export as html file
  • permission: consoleexporter.cmd
permissions:
  • name: consoleexporter.cmd
  • description: Allows use /consoleexport command - default: op

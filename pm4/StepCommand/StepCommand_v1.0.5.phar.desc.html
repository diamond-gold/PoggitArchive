StepCommand

Execute command when you step on a block!

NOTICE! If you had StepCommand on your server before, please delete all configuration files before downloading new version.

Permissions

stepcommand.bypass

Bypass command execution when player has this permission

Variables

{player} - Player's name

{world} - World the player is in

How To Use

  1. Download the plugin
  2. Put the file in your plugins folder
  3. Restart your server
  4. Go to plugin_data folder and find StepCommand
  5. Open config.yml and edit how you want
  6. Restart server again

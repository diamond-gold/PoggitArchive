PlayerWarpsUi this plugin to teleport to player warp easily

Commands

Usage

  • /pwarp (create, set, add) (pwarpName) - For create player warp
  • /pwarp (delete, del, remove) (pwarpName) - For delete player warp
  • /pwarp (newpos, new, edit) (pwarpName) - For edit new position player warp
  • /pwarp (info) (pwarpName) - Show information player warp
  • /pwarp (help) - Show help command
  • /pwarp (menu) - Show player warps menu ui

Features

  • Menu UI
  • Easy Config
  • BedrockEconomy Support
  • Custom Prefix

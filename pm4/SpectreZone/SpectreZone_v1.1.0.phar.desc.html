Intro

SpectreZone is a plugin based on a part of the Java Minecraft Random Things Mod

Features

  • The Spectre Dimension!
  • 3 Custom Items
    • Ectoplasm
    • Spectre Ingot
    • Spectre Key
  • 2 Custom Blocks
    • Spectre Dimension Wall Blocks
    • Dimension Growth Blocks
  • 3 new crafting recipes for added items

Future Additions

  • Spectre Mobs
  • Custom Spectre Biome
  • No others planned. Leave requests in the reviews on Poggit.

ClearLag v2.1.0

A pocketmine plugin, which removes entities after a certain period of time to avoid lags


|


Command

Command Description Permission
clearlag Removes entities to avoid lags op, clearlag.command

Features

✔️ Removes entities after a specified period of time
✔️ Configurable messages
✔️ Command to manually remove entities
✔️ Decide which entities should be removed
✔️ Allows to remove entities that do not inherit from Living
✔️ Allows to disable the ClearLag messages for the console

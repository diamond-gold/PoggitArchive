Welcome to the CustomItemAPI plugin

With this plugin, you can add any kind of item, pick, sword etc..

Why is it 'API'?

CustomItemAPI has been basic for developers, you can also find the wiki to start using the API in your plugins ! Although it is normally designed for developers, a small configuration has been set up to create basic items.

Can I add some food ?

Yes! The API allows it:)

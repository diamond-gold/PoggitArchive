AutoPickup

AutoPickup

|

A simple plugin that sends drops of mined blocks directly to the inventory.

Features

  • Sends blocks directly to the inventory
  • Sends xp directly to the player
  • Easy to use

Config

In the configuration file you can decide in which worlds AutoPickup should be activated. 

# Mode can be either "whitelist" or "blacklist"
# whitelist ensures that AutoPickup is only active in the worlds entered below
# blacklist ensures that AutoPickup is deactivated in the worlds entered below
mode: blacklist

# Set whitelist/blacklist worlds here
worlds:
  - placeholder

# Set a pop-up message here that appears when the player's inventory is full.
# Leave blank to disable pop-up messages
full-inventory: '&cYour inventory is full!'

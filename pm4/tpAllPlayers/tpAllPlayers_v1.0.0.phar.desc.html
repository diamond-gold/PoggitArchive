TpAllPlayers

The TPAllPlayers plugin is for teleporting all players on the server to you or your target! You can use the plugin completely at events, survival events and everything else. I will be happy if you write something in the comments that will help me for the better plugin. Thank you for your support and listen!

HowToUse

use /tpall for teleporting all players on the server to you

use /tpall (target) for teleporting all players on the server to target

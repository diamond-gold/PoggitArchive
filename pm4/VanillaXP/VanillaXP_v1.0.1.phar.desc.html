VanillaXP

Vanilla like xp command for your mcpe/mcbe PocketMine-MP server.

How to

Installation

  • Download phar file
  • Put the phar file in your plugins directory
  • restart your server
  • have fun

Adding XP/XP-LEVEL

  • /xp (amount)L (playername)
  • /xp (amount) (playername)

Removing XP/XP-LEVEL

  • /xp -(amount)L (playername)
  • /xp -(amount) (playername)

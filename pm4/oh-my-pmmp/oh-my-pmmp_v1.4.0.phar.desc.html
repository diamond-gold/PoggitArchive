General

Oh My PMMP provides commands to allow you to install plugins from Poggit. It's fast and easy to use.

Commands

Commands Description Aliases Permissions
/omp help List available subcommands h, -h, --help oh-my-pmmp.help
/omp version Get plugin version v, -v, --version oh-my-pmmp.version
/omp install Install a plugin
NOTE: To install the latest version, use "latest"
Example: /omp i ExamplePlugin latest		 i, -i, --install oh-my-pmmp.install
/omp remove Remove a plugin r, -r, --remove oh-my-pmmp.remove
/omp upgrade Upgrade a plugin u, -u, --upgrade oh-my-pmmp.update

NOTE: To use all commands of Oh My PMMP, use oh-my-pmmp.cmds permission.

Since RCON is no longer built into the core in 4.0.0, this plugin allows you to optionally support RCON again if you want to.

WARNING

Use of RCON is not recommended since it's insecure. Your passwords are sent over the internet in plaintext by RCON clients. Change your RCON password regularly.

Installing

  • Drop the plugin in your plugins folder and restart the server.
  • It will generate a rcon.yml file with default settings. If you want to customize the settings, stop the server, edit rcon.yml, and start the server again.

General

This plugin can automatically choose the best tool from your hotbar everytime you mine a block.

Features

  • Can Enable and Disable BestTools
  • Can blacklist any block you don't want
  • Can prevent your tool from being destroyed when destroying a block that doesn't match the tool type
  • Strip mining with random block ⛏️

Commands

Command Permission Description
/besttools besttools.use to Turn on or Off
/blacklist besttools.blacklist to blacklist some block

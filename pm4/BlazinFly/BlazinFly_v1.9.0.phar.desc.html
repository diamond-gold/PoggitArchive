Information

This plugin allows you to use /fly in survival mode to fly! Anyone is allowed to use this plugin.

  • Credits: BlazeTheDev

How to use

How to install:

  1. Download the plugin's phar.
  2. Drop the plugin's phar into the /plugins folder in your server
  3. Restart your server so that the plugin will be enabled!

How to use: Go in-game and run the command /fly in survival mode and you will be able to fly!

  • For more info on how to use this plugin go here BlazinFly Wiki.

DropProtection

A PocketMine-MP plugin which protects accidental drop

How does it work?

This plugin just prevents other players from picking up your items for a short amount of time, in case you have dropped something accidentally.

Commands

Command Description Usage Alias
/dropprotection Main Command /dropprotection /dropp
SubCommand Description
help View all subcommands
set <seconds> Set the drop protection duration
reload Reload the config file

NOTE: The drop protection duration must be a number between 0 and 300 excluded. 300 seconds (5 minutes) is the time which takes for a dropped item to despawn!

Config Setup

The config has the field dropProtectionDuration where you can define for how much time the dropped item are locked.

NOTE: The drop protection duration must be a number between 0 and 300 excluded. 300 seconds (5 minutes) is the time which takes for a dropped item to despawn!

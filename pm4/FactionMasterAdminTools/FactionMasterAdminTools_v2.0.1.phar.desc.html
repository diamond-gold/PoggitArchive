FactionMaster Invitation improve system

This plugin is part of FactionMaster and requires installation on the server, see here for more information

Propose a choice from a list of players or faction when an invitation is being created. For players, it will look at all registered players or those online. Improve the acceptance/sending system.

  • If a faction has already sent an invitation to a player and this one sends an invitation to the faction, it will automatically be accepted.
  • If a faction is public and that a player sends an invitation, it will be automatically accepted And much more !

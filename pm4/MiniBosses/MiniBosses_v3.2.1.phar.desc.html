MiniBosses

Adds very customizable Bosses

Encountered a bug? Want something added? Create an issue at Github

Reviews section will not be monitored!

Supported Entities

Theoretically any entity, not all entities tested

Features

Configurable

  • Attributes
    • Entity type
    • Health
    • Attack damage
    • Attack rate
    • Movement speed
    • Gravity
    • Scale
    • Hurt modifiers (Reduced arrow damage? No fall damage? No problem!)
    • Knockback resistance
  • Equipment
    • Held item
    • Offhand
    • Armor
  • Health display
  • Item/XP drops (option to spread drops)
  • Respawn time
  • Skin (Custom geometry supported)
  • Commands on death
  • Projectile
    • Entity type
    • Speed
    • Firing Range
    • Fire rate
    • Damage
    • Explosion radius
    • Health
    • Attacking/Deflecting
    • Despawn time
    • Gravity
    • Explosive flying pigs anyone? :P
  • Top damage rewards (items/commands)
  • Minions (Configured in the same manner as a Boss, can do anything a Boss can do)
  • Many more...

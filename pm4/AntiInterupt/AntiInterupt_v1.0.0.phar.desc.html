AntiInterupt

What is Antiinterupt?

AntiInterupt is a plugin use for combat. like if you pvp 1vs1 no one can interupt on you.

When you are in combat with a player other players is not your target can't fight on you.

Target will set after you fight on someone

If you fighting with player a and player b can't fight you while you in combat with a.

If you stop fighting in 8 second target will unset and send message to you.

Feature

  • Can set it on any world you want
  • Message :
    • Wrong target
    • Set target
    • Reset target.

How to use?

When you install plugin to server then go to folder

plugin_data/AntiInterupt and open file config.yml

config.yml :

 worlds: 
  - "world1"
  - "world2"

Add and edit world here to disable player interupt to another player while pvp

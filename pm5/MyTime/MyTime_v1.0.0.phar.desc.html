The MyTime plugin allows players to set their own virtual time in the game world independently of the server's time. Using commands such as /mtime day, /mtime night, and others, players can alter their personal time settings. This does not affect the server's time or other players' experiences. Additionally, players can reset their time to match the server's time with the /mtime reset command.

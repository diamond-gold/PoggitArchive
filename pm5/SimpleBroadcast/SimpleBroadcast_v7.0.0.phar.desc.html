SimpleBroadcast | v7.0.0

Automatic message system for PocketMine-MP, written in PHP-8 and only supports PocketMine-MP 5.0.0

Features:

  • Unlimited Messages
  • Add variables to messages
  • PocketMine-MP 5.0.0 (Only)

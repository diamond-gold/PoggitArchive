Make your network players switch between servers easily like proxy!

Please do not report bugs in the reviews. Always make sure that the issue in the plugin itself.

Usage

/server You can config the servers in ServerSwitcher/servers.yml

Configuration

servers:
  - name: "Server1"
    ip: "0.0.0.0"
    port: 19132
  - name: "Server2"
    ip: "0.0.0.0"
    port: 19133

name: Your server name that will be used in the command(/server <name>)
ip: Your server ip for transferring
port: Your server port for transferring

Features

  • Rank command have auto-completion
  • Flexible settings in config.yml
  • Local/Global chat
  • Rank inheritances
  • Rank priorities
  • Chat & Nametags included
  • Fast & Easy to use

API

  • Example:

    $player->sendMessage("Your rank: " . $rankManager->getRank($player)->getDisplayName());
}

Description

MaskUI is a pocketmine api 5 plugin to give players mob head items with an attractive ui. If you are have a question, please make a issues

Features

  • Custom message
  • Custom name
  • Custom sound effect
  • With permission
  • With economy (soon)
  • With effect (soon)

Installation

  1. Download the plugin
  2. Put plugin to plugins file
  3. Restart the server

Permission

Command Permission Default
/mask mask.ui OP
(give mask) mask.{type} OP

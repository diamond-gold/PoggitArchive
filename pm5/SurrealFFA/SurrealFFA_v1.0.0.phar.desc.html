SurrealFFA

SurrealFFA is a competitive and lightweight Free-For-All (FFA) plugin for PocketMine-MP, fully integrated with LibSurrealDB to store player statistics asynchronously in SurrealDB.

Features

  • Tracks kills and deaths per player
  • Real-time statistics saved in SurrealDB
  • Supports multiple FFA arenas
  • Uses LibSurrealDB for async database operations
  • Designed for high performance and scalability

Requirements

  • PocketMine-MP 5.0 or higher
  • SurrealDB running and accessible
  • LibSurrealDB virion or plugin installed

Commands

/ffa join [arena] – Join a FFA arena  
/ffa leave – Leave the FFA match  
/ffa stats [player] – View player stats

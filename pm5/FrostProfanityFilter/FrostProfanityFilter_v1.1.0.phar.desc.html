General

FrostProfanityFilter is a plugin designed for PocketMine-MP that enhances your server's chat environment by blocking profanities and unwanted words. This ensures a positive and respectful communication space for all players. Administrators can easily configure a list of banned words, making it adaptable to different community standards and preferences.

Features

  • Customizable Word List: Easily configure and update the list of words to be blocked in the chat.
  • Real-time Filtering: Automatically filters out unwanted words in real-time, maintaining a clean chat environment.
  • User-Friendly: Simple to install and configure, with minimal setup required to start filtering profanities.
  • Disable /me command: Plugin will automatically disable me command.

Installation

To get started with FrostProfanityFilter:

  1. Download the plugin from Poggit or the provided link.
  2. Place the FrostProfanityFilter.phar file in your PocketMine-MP server's plugins directory.
  3. Restart your server to activate the plugin.

Configuration

FrostProfanityFilter's configuration is straightforward:

  1. When you first run the plugin, it will generate a config.yml file in the plugin_data/FrostProfanityFilter directory.
  2. Open config.yml in a text editor to customize the list of blocked profanities.
  3. Each word or phrase to be blocked should be listed under the profanities key, with each entry on a new line.

Example config.yml:

profanities:
  - abortion
  - abuser
  - ahole
  - alabama hotpocket
  - alligatorbait
  1. After editing the config.yml file, save your changes and restart your server for the new settings to take effect.

Usage

Once configured, the plugin will automatically filter messages in the chat. Players attempting to send messages containing words from the blocked list will not be able to send them, thereby maintaining a positive chat environment.

This plugin is ideal for server administrators who want to enforce community standards and provide a friendly environment for all players.

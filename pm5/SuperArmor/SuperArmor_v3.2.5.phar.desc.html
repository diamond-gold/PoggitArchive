This is a plugin for buying armor with a high level of protection for diamonds or something else, setting through the config.

Command:

/armor

Description:

With this plugin, you can purchase armor for enchantment (protection 1000) for 64 diamonds. This armor protects well, but has very low durability! This plugin can be adjusted (cost, item-currency, enchantment level) using the config file (in plugin_data/SuperArmor/config.yml)

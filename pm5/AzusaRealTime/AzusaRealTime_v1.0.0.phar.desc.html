General

AzusaRealTime is a Pocketmine plug-in that works to always display real-world time on your server.

Features

  • Can set a country time of interest
  • can change the popup text as desired
  • There are 5 countries available. i.e. Indonesia, Malaysia, Singapore, India, and England.

Configuration

# Configuration for RealTime plugin

time: idn
popup-text: "§eTime: §f{hours}:{minute}:{second}"

# List of time for each country
# 'idn' for indonesian
# 'eng' for England
# 'ind' for India
# 'sgp' for Singapore
# 'mys' for Malaysia

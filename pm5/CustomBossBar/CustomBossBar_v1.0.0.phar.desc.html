CustomBossBar

A Pocketmine-MP plugin that will show a bossbar to each player that joins, Super simple and easy to use!

Everything is customizable!

Features

  • Customizable title/subtitle
  • Customizable colors
  • Customizable boss bar percentage

Virion/Dependency

You will need this virion for the plugin to work:

https://poggit.pmmp.io/ci/Terpz710/apibossbar/apibossbar

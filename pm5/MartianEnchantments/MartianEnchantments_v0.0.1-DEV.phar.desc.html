General

MartianEnchantments is a custom enchantment system for PocketMine-MP, inspired by plugins like AdvancedEnchantments, but rewritten from scratch for modern PocketMine versions.

It allows server owners to create, modify, and manage custom enchantments entirely through configuration, without relying on vanilla or PocketMine’s built-in enchantment system.

Key Features

  • Fully config-driven enchantments (enchantments.yml)
  • Create, edit, or remove enchantments without code changes
  • Custom triggers such as ATTACK, DEFENSE, HELD, EFFECT_STATIC, and more
  • Per-level chance, cooldowns, conditions, and effects
  • Supports multiple effects per enchantment level
  • Uses a custom NBT-based enchantment system
  • Not tied to vanilla or PM enchant mechanics

Triggers & Effects

Enchantments activate through triggers that define when they run (e.g. attacking, defending, holding an item, wearing armor).

This version includes 9 implemented effects:

  • ACTION_BAR
  • ADD_AIR
  • ADD_FOOD
  • ADD_HEALTH
  • ADD_POTION
  • BLOOD
  • BURN
  • DISABLE_ACTIVATION
  • STEAL_HEALTH

More triggers and effects can be added in future updates.

Commands & Items

The plugin includes multiple enchantment-related commands.

Some enchantment-related items exist, but full item functionality is still being expanded and will be improved in upcoming releases.

For Developers

MartianEnchantments includes utility methods intended for developers, such as:

  • Custom enchantment management
  • Enchantment extraction from items
  • Trigger execution helpers

This makes it easier to integrate or extend the enchantment system.

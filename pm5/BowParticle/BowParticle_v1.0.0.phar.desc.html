BowParticle

A plugin for pocketmine 5 that shows particles when shooting arrows!

Features

  • UI menu for easy particle selection
  • Customizable ticks between each particle
  • Customizable prefix

Particles

  • Flame
  • Water
  • Heart
  • Smoke
  • Portal
  • LavaDrip
  • WaterDrip
  • Redstone
  • SnowballPoof
  • AngryVillager
  • HappyVillager
  • EnchantmentTable

Command

Command Description Permission Default
/bowparticle Select a particle for your bow shots! bowparticle.command op

Description

AquaCrates is a plugin that allows users create unlimited crates and customize almost everything.

Command

  • /crate create string:name - Create a crate
  • /crate addkey string:player string:crate [int:amount] - Add crate keys to a player
  • /crate removekey string:player string:crate [int:amount] - Remove crate keys from a player
  • /crate set|replace string:crate - Change the position of a crate
  • /crate keyall string:crate [int:amount] - Give online players crate keys

Permission

  • aquacrates.permission

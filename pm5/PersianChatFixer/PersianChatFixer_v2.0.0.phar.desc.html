PersianChatFixer is a high-performance plugin for PocketMine-MP that fixes Persian (Farsi) text rendering issues in Minecraft Bedrock servers. It ensures that Persian messages are displayed with correct direction and alignment in both chat and signs, while handling mixed English content and punctuation seamlessly.

Features

  • ✅ Accurate right-to-left (RTL) display for Persian text
  • ✍️ Correct alignment and formatting on Minecraft signs
  • 🔁 Smart handling of mixed Persian-English messages
  • 🧠 Intelligent bracket and punctuation mirroring (e.g. parentheses, question marks)
  • ⚡ Optimized for low CPU usage, ideal for high-performance servers
  • 🔧 Works out-of-the-box with zero configuration

How It Works

PersianChatFixer intercepts all chat and sign messages, identifies the dominant language direction, and applies bidirectional formatting rules. It uses Unicode-aware logic to:

  • Reverse the visual order of Persian characters
  • Maintain English word order inside Persian text
  • Flip punctuations like ( to ) only when needed
    This results in a natural and readable chat/sign output for Persian-speaking players.

Compatibility

  • ✅ PocketMine-MP API versions: 5.14.0 – 5.30.1
  • 🧩 PHP 8.1+ (fully tested)
  • 🌐 Supports all Minecraft Bedrock clients

Use Cases

  • Persian-speaking Minecraft communities
  • Creative and roleplay servers with multilingual chat
  • Servers that aim to support RTL languages natively

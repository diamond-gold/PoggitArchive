CustomRankJoinMessage is a plugin designed to display personalized messages when players join your server, based on their rank. This plugin allows server administrators to customize the welcome messages based on player ranks, enhancing the player experience by showing unique, rank-specific greetings. It seamlessly integrates with the RankSystem plugin for automatic rank detection.

Features

  • Rank-Specific Join Messages: Display different join messages for players based on their rank (e.g., VIP, Admin, Default).
  • Full Customization: Edit join messages with custom text, colors, and formats to suit your serverâ€™s style.
  • RankSystem Integration: Manually add ranks and their corresponding custom join messages to the config.yml. If a player's rank matches one in the configuration, the custom join message will be displayed. Otherwise, the default join message will be shown.
  • Lightweight and Fast: Optimized for performance, this plugin will not put a strain on your server.

Installation

  1. Download the .phar file for CustomRankJoinMessage from this page.
  2. Upload the file to your plugins/ directory on your PocketMine-MP server.
  3. Open the config.yml file and adjust the messages as needed.
  4. Reload or restart the server for the changes to take effect.

Configuration Example

Below is an example of a config.yml file that you can use to customize your join messages:

join-messages:
  FirstRankName: "Custom first join message!"
  SecondRankName: "Custom second join message!"

leave-messages:
  FirstRankName: "Custom first leave message."
  SecondRankName: "Custom second leave message."

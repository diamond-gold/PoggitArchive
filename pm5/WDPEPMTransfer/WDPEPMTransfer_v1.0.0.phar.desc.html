WDPEPMTransfer

WaterdogPE transfer support for PocketMine-MP.

Command Description Usage
/transfer Transfer to a another server. For player (in-game): /transfer <server_name> [player_name] [port]
For console: /transfer <player_name> <server_name> [port]

Installation

This is a plug-and-play plugin. All you need to do is the following:

  1. Download WaterdogPE from github releases.
  2. After that, make sure you also installed Pocketmine-MP and downloaded the plugin from poggit builds or releases.
  3. Thats all! Just run /transfer command to transfer another server!

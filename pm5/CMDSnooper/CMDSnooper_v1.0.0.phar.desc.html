Feature

  • This is plugin Allows admin or console to observe players using commands on the server

Command & Permission

Command Description Permission
/snooper Watch players use commands cmdsnooper.command

Config

Console.Logger: "true"
#change if you don't want the console to show up

Install

  • Step 1: Click the Direct Download button to download the plugin
  • Step 2: move the file CMDSnooper.phar into the file plugins
  • Step 3: Restart server

AntiBugPing is a lightweight plugin that detects and mitigates abnormal network behavior such as lag-switching and desynchronization in PocketMine-MP 5 servers.

Features

  • Real-time monitoring of player connection (ping, jitter, packet flow)
  • Dynamic risk scoring with automatic decay to reduce false positives
  • Progressive mitigation system based on network instability
  • Optional actions: restrict combat, freeze movement, setback, or kick
  • No database required, fully in-memory

Performance

AntiBugPing is designed with performance in mind:

  • Zero database usage
  • Event-driven execution (no heavy repeating tasks)
  • Optimized memory handling using efficient data structures

Configuration

The plugin is fully configurable:

  • Adjust ping and jitter thresholds
  • Control mitigation sensitivity
  • Configure watchdog behavior
  • Enable or disable kick system

It is recommended to tune the configuration based on your server network conditions.

Permissions

  • antibugping.bypass → Ignore all checks
  • antibugping.notify → Receive alerts and debug information

Usage

Simply install the plugin and adjust the configuration file if needed.
The system will automatically start monitoring players and applying mitigation when necessary.

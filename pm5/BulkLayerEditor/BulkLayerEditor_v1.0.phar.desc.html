Features

Remove blocks in otherwise inaccessible blocklayers.
This allows for proper uninstallation of Plugins which interact with the fluid layer (e.g. implement water or snowlogging)

Usage

To use the tool, simply drop the phar into the plugins folder and restart the server.
Then edit the config.yml to your liking and restart the server again. Worlds should now slowly be converted (This can take a while depending on the size of the world).

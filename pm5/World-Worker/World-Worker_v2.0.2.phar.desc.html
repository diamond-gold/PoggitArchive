General

This plugin is designed for managing worlds in PocketMine-MP. With this plugin, you can create new worlds with generation options and perform various operations on worlds. The plugin does not require any external plugins to run.

Requirement:

  • PocketMine-MP 5.0.0 or higher

List of operations:

  • generate (create a world)
  • rename (rename a world)
  • delete (delete a world)
  • load (load a world)
  • unload (unload a world)

Command:

  • worker

Permission

  • worker - world-worker.worker //open operation list

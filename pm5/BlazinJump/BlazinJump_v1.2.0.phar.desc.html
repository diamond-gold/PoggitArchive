Information

This plugin is a double jump plugin. It allows the player when they jump twice to be launched into the air! This plugin is for anyone to use. If you have any issues with the plugin, please make an issue here.

  • Credits: BlazeTheDev and MCPEExpert

How to use

How to install:

  1. Download the plugin's phar from here
  2. Drop the plugin's phar into the /plugins folder in your server
  3. Restart your server so that the plugin will be enabled!

How to use: Go in-game and jump twice and you will be launched into the air!

For more info about how to use the plugin go to the BlazinJump Wiki.

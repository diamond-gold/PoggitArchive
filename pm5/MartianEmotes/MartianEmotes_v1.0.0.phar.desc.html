MartianEmotes

MartianEmotes adds a fun and customizable way for players to use emotes in chat on your Minecraft server. This plugin allows players to trigger custom emotes by typing specific phrases, which are then replaced with corresponding emote outputs.

Features

  • Customizable Emotes: Create your own emotes by defining custom phrases and their corresponding outputs.
  • Permissions-based Control: Control who can use each emote with the built-in permissions system.
  • Paginated Emote Listing: Display emotes in an easy-to-browse paginated list via the /emotes command.
  • Fully Configurable: Configure emotes and settings to your liking via the config.yml file.
  • Color Support: Emotes and messages support Minecraft’s color codes for better visual appeal.

Usage

In Chat

Players can use emotes by typing the associated phrases in the chat. For example:

  • Typing :wave: will be replaced by (•◡•)/.
  • Typing :heart: will be replaced by ❤.

Permissions

  • By default, all players can use emotes.
  • If the all-default option in the configuration is set to false, only players with the permission martian_emotes.<identifier> can use specific emotes.

Commands

/emotes [page]

This command allows players to view a paginated list of available emotes. Use the page argument to view specific pages.

Example: /emotes 2 will show the second page of emotes.

Configuration

The plugin is fully configurable via the config.yml file. You can:

  • Define custom emotes with specific phrases and outputs.
  • Set whether all players can use emotes or restrict them using permissions.
  • Control the number of emotes displayed per page in the command listing.

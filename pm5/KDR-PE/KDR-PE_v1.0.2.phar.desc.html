KDR-PE for Pocketmine-MP

Description

A Pocketmine-MP plugin that adds a KDR managment system.

This plugin creates a floating text leaderboard that shows off the top kills on the server as well!

Features

  • Kill tracker
  • Death tracker
  • KDR system
  • Topkill system
  • See other players KDRS
  • Scorehud support
  • CREATE A TOP KILL LEADERBOARD(Floating Text).

ScoreHud Tags

  • Kills: {kdrpe.kills}
  • Deaths: {kdrpe.deaths}
  • KDR: {kdrpe.kdr}

Link to his plugin: ScoreHud

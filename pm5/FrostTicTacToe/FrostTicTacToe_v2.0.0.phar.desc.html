General

FrostTicTacToe is an engaging Pocketmine MP plugin that introduces the popular TicTacToe game to your Minecraft server, providing players with a fun and competitive gaming experience.

Features

Interactive Gameplay

FrostTicTacToe allows players to initiate, join, and manage TicTacToe games through simple in-game commands.

Command-Based Controls

Players can use commands such as /fttt start, /fttt join "GAMERTAG", /fttt list, and others to seamlessly interact with the TicTacToe game.

Installation

  1. Download the latest .phar release from the Poggit Release Page.
  2. Place the downloaded file into your Pocketmine MP server's plugins directory.
  3. Restart your server to load the plugin.

Changelog v2.5.0

Fixes

  • Fixed kit item ID parsing issues
  • Various bug fixes

New Features

  • Added Kills, Deaths, and Kill Streak leaderboards
  • Arena-specific kits can now be set
  • New commands: /ffa setkit, /ffa spawntop, /ffa removetop
  • Optional kit parameter in arena creation: /ffa create [kitName]

API Updates

  • Added API::getKillStreak() function
  • Added API::getTops() function
  • Added API::getPlayerInfo() function

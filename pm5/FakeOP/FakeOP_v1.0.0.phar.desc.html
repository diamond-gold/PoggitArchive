FakeOP:

Fake OP is a plugin designed for a funny troll, simply because it looks legit and makes people really believe they have been opped.

Usage:

the only command in this version is /fakeop e.g: /fakeop skyss0fly

Permissions:

The permission to fakeop is FakeOP.use

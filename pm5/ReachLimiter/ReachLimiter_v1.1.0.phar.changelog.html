📝 What's New in v1.1.0 (The Intelligence Update)
This update completely rewrites the reach validation logic to introduce Smart Detection. ReachLimiter now understands that not all hits are equal, especially when lag is involved.

Changelog:

📶 Added Ping Compensation: The plugin now dynamically adjusts the maximum reach based on the player's latency. No more false positives for high-ping players!

🛡️ Enhanced Buffer System: Introduced extra-buffer to handle natural movement jitter and player velocity.

🌍 World Support: You can now whitelist/blacklist specific worlds. Perfect for servers with non-PvP areas.

📢 Advanced Staff Alerts: Staff members now receive detailed real-time data including the player's ping, the exact distance, and the world where the violation occurred.

⚡ Logic Optimization: The detection engine was rewritten to be even more lightweight, using MONITOR priority for better compatibility with other plugins.

⚙️ Config v2: A more organized config.yml with deeper control over the checking engine.

Bug Fixes

  • Fixed possible server crashes when changing module states during runtime (i.e., /invseemod enable <module> and /invseemod disable <module>)

API Changes

  • Dropped various getter methods in favor of direct property access
    • InvSeePlayer::getPlayer() -> InvSeePlayer::$player
    • InvSeePlayer::getInventoryMenu() -> InvSeePlayer::$inventory_menu
    • InvSeePlayer::getEnderChestInventoryMenu() -> InvSeePlayer::$ender_inventory_menu
    • InvSeePlayer::getLogger() -> InvSeePlayer::$logger
    • OfflinePlayerInventory::getOfflinePlayerData() -> OfflinePlayerInventory::$data
    • EnderInvSeeCommandExecutor::getViewPermissionChecker() -> EnderInvSeeCommandExecutor::$view_permission_checker
    • InvSeeCommandExecutor::getViewPermissionChecker() -> InvSeeCommandExecutor::$view_permission_checker
    • ModuleCommand::getName() -> ModuleCommand::$name

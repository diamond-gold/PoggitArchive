VBackpack

- A plugin for PocketMine-PMMP 5. - Bringing newness and utility to features for your server becomes diverse and interesting.

Features

  • Put everything in your backpack without filling your bags.

How to install it ?

  • You should install InvMenu here: Sources

Commands

/vbackpack or /vbp - give <player> <type> <amount>

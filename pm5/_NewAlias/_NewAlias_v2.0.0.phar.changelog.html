End-User Changes

  • Data Storage: Player data is now stored in a new, more organized format. Old data will be automatically migrated to the new format.
  • Configuration: The config.yml file has been updated with a new structure. Please review the new configuration options.

Developer API Changes

  • Data Hashing: All data saved to the database, except for usernames, is now hashed using SHA512.
  • Database Schema: The database schema has been updated to support the new data hashing. The Players table has been replaced with Addresses,
    ClientRandomIds, DeviceIds, SelfSignedIds, and XUIDs. The columns that store hashed data have been changed to VARCHAR(128).
  • Asynchronous Operations: Database operations are now fully asynchronous, using await-generator.
  • Configuration: The plugin now uses a structured configuration system with libMarshal.
  • API Version: The plugin API version has been updated to 5.0.0.
  • Dependencies: The plugin now uses composer to manage dependencies.
  • CI/CD: The plugin now uses GitHub Actions for CI/CD.
  • Removed .poggit.yml: The .poggit.yml file has been removed in favor of GitHub Actions.
  • Updated plugin.yml: The plugin.yml file has been updated with the new API version and dependencies.

New Database Class

The JavierLeon9966\Alias\database\Database class is new and provides the following public methods for interacting with the database:

  • public static function create(RawQueries $queries): Generator
  • public function addAddress(string $username, string $address): Generator
  • public function addClientRandomId(string $username, string $clientRandomId): Generator
  • public function addDeviceId(string $username, string $deviceId): Generator
  • public function addSelfSignedId(string $username, string $selfSignedId): Generator
  • public function addXuid(string $username, string $xuid): Generator
  • public function getPlayersMatchingAddressesFrom(string $username, ?string $extraAddress = null): Generator
  • public function getPlayersMatchingClientRandomIdsFrom(string $username, ?string $extraClientRandomId = null): Generator
  • public function getPlayersMatchingDeviceIdsFrom(string $username, ?string $extraDeviceId = null): Generator
  • public function getPlayersMatchingSelfSignedIdsFrom(string $username, ?string $extraSelfSignedId = null): Generator
  • public function getPlayersMatchingXUIDFrom(string $username, ?string $extraXuid = null): Generator

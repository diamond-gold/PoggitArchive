WorldBorder plugin for Pocketmine-MP

Description

Add a world border to your Pocketmine-MP server with BorderLimitPE! With this plugin you can configure a world border within the config.yml to set a world border with a min and max radius.

Features

  • World border for each world
  • Easy configuration
  • Configurable messages
  • Players gets pushed back if they pass the world border

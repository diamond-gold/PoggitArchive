FancyPlayerCount ✨

Fancy Player Count Inspired by Zeqa Network

Feature 💥

This sets the your pocketmine player count (current count / current count + 1) which gives value of (100/101). 📌

⚠️ This plugin does not modifies player count internally. This plugin changes only in the Query. This plugin is not intended for bypassing max player count.

How to use? ⚙️

  • Just download this plugin from poggit.
  • Drop to your plugins folder at your PocketMine-MP Server
  • Run the server and enjoy!

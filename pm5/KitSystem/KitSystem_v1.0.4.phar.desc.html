🧰 KitSystem – The Ultimate Kit Management Plugin

KitSystem is a modern, lightweight, and fully-featured kit management plugin for PocketMine-MP. Designed with performance and scalability in mind, this plugin allows server administrators to create, edit, categorize, and manage kits with ease using interactive forms.

💡 Key Features:

  • 📦 Form-based interface – Create and manage kits and categories easily through interactive forms.
  • 🗂️ Category support – Organize your kits into categories for better navigation.
  • 💾 SQLite & MySQL support – Choose the best storage option for your server.
  • 💰 Multi-economy compatible – Integrates with all economy plugins supported by libPiggyEconomy.
  • 🌐 Multi-language support – Easily translate and localize your kit system.
  • 🛠️ Advanced editing tools – Modify kits and categories with just a few clicks.
  • 🎁 Starter kits – Optionally give kits to new players on first join.
  • 🔧 Highly customizable – Messages, permissions, and forms can all be adjusted to your needs.
  • 🔍 Kit previews – Let players view the contents of a kit before claiming it.
  • 🧼 Clean, robust codebase – Written to pass PHPStan at max level for maximum reliability.

📋 Commands:

  • /kit – View available kits
  • /kit create – Create a new kit
  • /kit delete – Delete an existing kit
  • /kit give – Give a kit to a specific player
  • /kit giveall – Give a kit to all players
  • /kit createcategory – Create a new category
  • /kit deletecategory – Remove a category
  • /kit edit – Edit an existing kit
  • /kit editcategory – Edit a category
  • /kit preview – View the contents of a kit

📚 Dependencies:

  • Commando
  • EasyUI
  • InvMenu
  • item-serialize-utils
  • libPiggyEconomy
  • libasynql
  • Languages
